# LawnStringsData

Representa los datos de un archivo *LawnStrings*, almacenando la lista de cadenas localizadas por idioma.

## Constructores

▶ `LawnStringsData()`

Crea una nueva instancia de la clase `LawnStringsData`, inicializando la estructura básica sin valores asignados.

### Ejemplo

```csharp
LawnStringsData data = new LawnStringsData();
```

## Propiedades

▶ `LocStringValues`

Lista de cadenas localizadas según el idioma.
