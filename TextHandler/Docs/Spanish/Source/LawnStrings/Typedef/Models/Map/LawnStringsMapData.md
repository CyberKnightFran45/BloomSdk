# LawnStringsMapData

Clase que contiene los strings localizados por idioma.

## Constructores

▶ `LawnStringsMapData()`

Crea una nueva instancia de la clase `LawnStringsMapData`.

### Ejemplo

```csharp
LawnStringsMapData data = new LawnStringsMapData();
```

## Propiedades

▶ `LocStringValues`

Obtiene o establece los strings localizados por idioma.
